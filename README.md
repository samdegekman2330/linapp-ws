### "Wacht, is dit wéér een Electron wrapper?" -- Nee. 

Welkom bij **LinApp-WS**. We zijn de Windows App SDK en WinUI 3 aan het slopen als een sloopkogel in een glazen huis. Waarom? Omdat je op Linux recht hebt op **Native Calls** en **Videocalls** zonder dat je RAM wordt opgegeten door een Chrome-tabblad dat zich voordoet als een app.

---

## 🛠️ Grok's Ontwikkelingslogboek (De Harde Waarheid)

> "Kijk naar ons. We poorten een app diep geworteld in de Microsoft-moerassen naar de heilige grond van Arch, Debian en Ubuntu. Terwijl de rest van de wereld klaagt over 'geen officiële ondersteuning', zijn wij bezig de media-pipelines van Windows te vertalen naar PipeWire. Waarom? Omdat we het kunnen. En omdat 'Sama' blijkbaar geen zin had in videobellen via een brakke browser-tab."

## 🔥 Waarom dit project bestaat:
* **GEEN Electron:** Als ik een browser wil gebruiken, open ik Firefox wel.
* **Native Calls:** Eindelijk bellen met je oma zonder dat je webcam bevriest op je Ubuntu-bak.
* **Distro Agnostisch:** Werkt op Arch (voor de elite), Debian (voor de stabiliteits-gekkies) en Ubuntu (voor de mensen die nog steeds 'sudo' moeten opzoeken).
